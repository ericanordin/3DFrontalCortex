Code to make 3d reconstuctions of electrode locations in the rat brain based on Paxinos & Watson's atlas. 
Originally written for Swanson's atlas; modified for P&W with extended brain regions by Erica Nordin.
This is code in progress, and requires command of matlab and knowledge of rat anatomy. 

Written by Eyal Kimchi, modified by Kumar Narayanan and Erica Nordin.
